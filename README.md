## なぜこの企画設計にしたか

本提案書は **「実店舗の格を、画面の中でも保つ」** ことを唯一の設計指針として組み立てました。

恵比寿の高級焼肉店は **接待・記念日・デート** 利用が中心で、ユーザーは常に「失敗できない夜」という文脈で店を選びます。だからこそWebサイトは、料金や雰囲気が言葉で伝わるだけでは足りず、画面そのものが店の照明設計と同じ温度を持つ必要がある。これが企画の出発点です。

提案書自体も同じ思想で組みました。

- **タイポグラフィ** — Cormorant Garamond × 游明朝の和洋ペアリング
- **配色** — 漆黒（#0A0805）と古金色（#C9A84C）のミニマル構成
- **レイアウト** — 編集デザイン的な広い余白、グリッドの厳密な統一

30ページ全てを「美術館の図録」の文体で統一し、**ビジネス課題への解答としてのデザイン** を最後まで一貫させています。

## 30ページ構成

### 戦略・企画（P.03–09）
プロジェクト概要、KPI設計、ペルソナ設計、競合分析、課題の根本原因とビジネスインパクトの定量化

### デザイン設計（P.10–12）
サイトコンセプト、ビジュアルランゲージシステム、「なぜそうしたか」の意図開示

### 情報設計・導線（P.13–22）
サイトマップ、ワイヤーフレーム、UI/UX設計、レスポンシブの再設計思想、予約導線とCTA階層、空席予約UI、Googleマップ導線、制作で工夫した6つの細部

### 技術・振り返り（P.23–30）
SEO 7レイヤー戦略、使用技術、スケジュール、実務ポイント、今後の改善案、トップページ表示イメージ、公開リソース（QRコード）

## 使用技術

| 区分 | 技術 |
|---|---|
| マークアップ | HTML5（セマンティック構造、アクセシビリティ配慮） |
| スタイリング | CSS3（CSS変数によるデザイントークン一元管理、`@media print` 対応） |
| 欧文タイポグラフィ | Cormorant Garamond / DM Sans（Google Fonts、`preconnect` で初期描画最適化） |
| 和文タイポグラフィ | 游明朝（システム搭載フォントスタック） |
| ホスティング | GitHub Pages（SSL自動、CDN配信） |
| ビルドツール | なし（フレームワーク・ビルド不要のバニラ構成） |

PDF出力は印刷時のメディアクエリ（`@media print`）で1280×900px相当に最適化されており、ブラウザの印刷機能からそのままA4横相当のPDFを生成できます。

## 閲覧方法

### Web上で閲覧
GitHub Pagesで公開済みです。下記URLからそのまま閲覧できます。

```
https://hirotonozaki.github.io/yakiniku-kokuyou-proposal/
```

### ローカルで閲覧
```bash
git clone https://github.com/hirotonozaki/yakiniku-kokuyou-proposal.git
cd yakiniku-kokuyou-proposal
open index.html
```

ビルド不要のバニラ構成のため、`index.html` をブラウザで開けばそのまま閲覧できます。

### PDFとして書き出す
ブラウザの印刷機能から以下を選択してください。

| 項目 | 値 |
|---|---|
| 保存先 | PDFに保存 |
| 用紙サイズ | A4 横 |
| 余白 | なし |
| 背景画像 | オン |
